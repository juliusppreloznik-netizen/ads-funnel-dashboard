# Transcript Analyzer Skill

## When to use this skill
Trigger whenever the user mentions: analyzing transcripts, extracting insights from calls, building creative briefs from sales data, mining calls for copy angles, or anything related to sales call intelligence gathering.

## Workflow
Ingest → Extract (Phase 1) → Analyze Patterns (Phase 2) → Output Creative Brief (Phase 3)

## Critical Rule
ALL extracted language must be verbatim — never cleaned up, never paraphrased, never made "professional." The raw language IS the weapon.
